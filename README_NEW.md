Yachi — Skilled Workforce Marketplace

This README is a concise, general guide for developers, maintainers and operators. It complements `README_PROFESSIONAL.md` (detailed deployment guidance) and `feedback.md` (audit & readiness checklist).

Quick links:
- `backend/` — API server
- `frontend/` — Web client
- `mobile/` — Mobile app

Getting started:
- Install: `npm ci` at repo root
- Backend: `cd backend && copy .\.env.example .\.env && npm ci && npm run dev`
- Frontend: `cd frontend && npm ci && npm run dev`
- Mobile: `cd mobile && npm ci && npm start`

Testing & linting:
- Run tests and linters in each workspace. See `CONTRIBUTING.md` for details.
